Resource Resolver

Resource Resolver is a way to resolve request attributes and get directly the model.

It is very similar to Param Converters. They are, however far more flexible and powerful.

{% highlight php %} getDoctrine() ->getManager() ->getRepository('App:Blogpost') ->findAll() ; // your logic } public function showAction($blogPostId) { $blogPost = $this->getDoctrine() ->getManager() ->getRepository('App:Blogpost') ->find($blogPostId) ; // your logic } } {% endhighlight %}
